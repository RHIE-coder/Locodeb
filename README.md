# Convention

 - class `class CamelCase {}`
 - function `function doSomething() {}`
 - constant `const UNDERLINE_SNAKE='value'`
 - variable `let changableVar='value'`
 - parameter(not argument) `_inputArgs`
 - application/json `{ "key_for_transport":"value" }`
